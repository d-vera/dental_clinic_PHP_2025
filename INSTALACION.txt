📋 REQUISITOS PREVIOS
─────────────────────
✓ XAMPP instalado (Apache + MySQL + PHP)
✓ Navegador web (Chrome, Firefox, Edge, etc.)

🚀 PASOS DE INSTALACIÓN
─────────────────────

1️⃣ INICIAR XAMPP
   - Abrir XAMPP Control Panel
   - Click en "Start" para Apache
   - Click en "Start" para MySQL
   - Esperar a que ambos estén en verde

2️⃣ CREAR LA BASE DE DATOS
   - Abrir navegador
   - Ir a: http://localhost/phpmyadmin
   - Click en "Nueva" (o "New")
   - Nombre de la base de datos: clinica_medica
   - Cotejamiento: utf8mb4_unicode_ci
   - Click en "Crear"

3️⃣ IMPORTAR LOS DATOS
   - Seleccionar la base de datos "clinica_medica"
   - Click en la pestaña "Importar" (o "Import")
   - Click en "Seleccionar archivo"
   - Buscar y seleccionar: database.sql
   - Click en "Continuar" o "Go"
   - Esperar mensaje de éxito

4️⃣ ACCEDER AL SISTEMA
   - Abrir navegador
   - Ir a: http://localhost/fourthBim/4. final_project/
   - ¡El sistema está listo!

🐛 SOLUCIÓN DE PROBLEMAS
─────────────────────────

❌ Error: "No se puede conectar a la base de datos"
   ✓ Verificar que MySQL esté corriendo en XAMPP
   ✓ Verificar que la base de datos "clinica_medica" existe
   ✓ Revisar config/database.php

❌ Error: "Página no encontrada" o 404
   ✓ Verificar la ruta: debe ser fourthBim/4. final_project/
   ✓ Verificar que Apache esté corriendo
   ✓ Revisar que la carpeta esté en htdocs

❌ Las imágenes no se guardan
   ✓ Crear manualmente las carpetas:
      - uploads/pacientes/
      - uploads/medicos/
   ✓ Dar permisos de escritura si es necesario

❌ No puedo iniciar sesión
   ✓ Verificar que importaste database.sql
   ✓ Usar exactamente: admin / admin123
   ✓ Verificar que las cookies estén habilitadas
